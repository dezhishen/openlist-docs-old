GET 列出元信息

GET /api/admin/meta/list

请求参数

名称位置类型必选说明
pagequerystring否页数
per_pagequerystring否每页个数
Authorizationheaderstring是none

返回示例

成功

{
  "code": 200,
  "message": "success",
  "data": {
    "content": [
      {
        "id": 1,
        "path": "/a",
        "password": "i",
        "p_sub": false,
        "write": false,
        "w_sub": false,
        "hide": "",
        "h_sub": false,
        "readme": "",
        "r_sub": false
      }
    ],

}
